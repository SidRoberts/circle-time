# Circle Time

## Usage

Access via [https://sidroberts.co.uk/circle-time](https://sidroberts.co.uk/circle-time).

### Docker

```bash
docker build . -t sidroberts/circle-time

docker run --rm -p 80:80 sidroberts/circle-time
```
